Решение от Александра Титова 
Тестовое задание от Avito для data sciense bootcamp

# О задаче
Пользователи Авито обращаются в поддержку с короткими вопросами:

- как отправить товар покупателю,
- почему объявление всё ещё на проверке,
- как оформить возврат,
- где посмотреть статус заказа,
- что делать, если не проходит оплата.
В продуктовой системе такие статьи не используются как готовый ответ пользователю. Они служат документами для RAG-пайплайна: сначала нужно найти подходящие материалы, а уже потом на их основе формируется ответ.

В этой задаче нужно решить этап поиска: по тексту вопроса пользователя найти статьи, которые стоит передать дальше в пайплайн.

# Цель задания
Разработать решение, которое:

1. Принимает текст пользовательского вопроса.
2. Ищет релевантные документы в базе статей справочного центра.
3. Возвращает ранжированный список article_id: сначала самые подходящие документы, затем менее подходящие.
   
В одном запросе может быть несколько правильных статей. Поэтому важно не только найти одну наиболее похожую статью, но и расположить все релевантные документы как можно выше в топ-10.
